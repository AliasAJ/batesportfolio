G-Books
Find your favorite books using the Google Books API
Description :
G-Book is a book search application that uses the Google Books API to provide users the title, subtitle, cover, and brief description of books they search for. The search can take book titles or keywords to help find the match users are looking for. Upon clicking the book cover, users are redirected to the books page on the Google Books website.
